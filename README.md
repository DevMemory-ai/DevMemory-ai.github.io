# DevMemory AI Documentation Source

This directory contains the source files for the official DevMemory AI interactive documentation and landing page hosted at [devmemory-ai.github.io](https://devmemory-ai.github.io/).

## Files

- `index.html`: The main documentation single-page application. Features a built-in terminal simulator, clean responsive sidebar navigation, and detailed command reference documentation.
- `app.js`: Houses the simulated terminal animation logic that runs on the landing page hero section.
- `style.css`: Contains the premium styling (dark mode, glassmorphism, responsive grid layouts) for the docs.
- `parkospace_ad.jpg`: Sponsored banner asset displayed in the landing page modal.
